# Financial sight – Personal Finance Tracker

A desktop GUI application built with Python (Tkinter) for tracking personal finances.

## Features
- Add, update, delete transactions (CRUD)
- Filter by Income/Expense
- Search transactions
- Real-time balance summary
- Pie chart visualization (Matplotlib)
- Export data to CSV

## Tech Stack
- Python
- Tkinter
- SQLite
- Matplotlib

## How to Run
```bash
python -m ui.app
